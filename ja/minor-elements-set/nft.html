NFT

Non-Fungible Token の略称であり、 ブロックチェーン技術を活用して代替不可能な価値をトークン化したものです。 BizenDAOでは、 指紋のように一つひとつ異なる備前焼の「自然釉」の景色を鮮明な写真や3Dモデルとして NFTのメタデータに含めることで、代替不可能な備前焼作品の証明書として利用しています。 また、作品のトレーサビリティ（来歴）としても活用しています。
